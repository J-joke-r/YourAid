# YourAid

## Description
YourAid is a project designed to help users find and share random jokes.

## Features
- A user-friendly interface for browsing jokes
- Ability to add new jokes
- Search functionality for finding specific jokes

## Technologies Used
- JavaScript
- React
- Node.js
- Express
- MongoDB

## Installation
To install the project locally:
1. Clone the repository:
   ```bash
   git clone https://github.com/imanishraj/YourAid.git
   ```
2. Navigate to the project directory:
   ```bash
   cd YourAid
   ```
3. Install the dependencies:
   ```bash
   npm install
   ```

## Usage Instructions
1. Start the application:
   ```bash
   npm start
   ```
2. Open your browser and go to `http://localhost:3000`.
